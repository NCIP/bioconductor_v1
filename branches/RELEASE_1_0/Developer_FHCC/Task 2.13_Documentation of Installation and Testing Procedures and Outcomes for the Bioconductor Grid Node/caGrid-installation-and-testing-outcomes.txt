Installation issues
-------------------

- No special installation issues

Deployment issues: caGrid
-------------------------

- Service deployed to Tomcat on port 80, rather than 8080. This
  requried reconfiguing Tomcat and our integration client, but no
  apparent issues with deployed services.

- Multiple services deployed to the same Tomcat container, without
  issues with class name conflicts experienced by others.

Deployment issues: RWebServices toolchain
-----------------------------------------
  
- activeMQ version mismatches on server and workers required
  harmoniazation to activeMQ v.4.0.2

- Non-standard port configurations (due to security concerns) required
  adjustment of service configuration parameters (in the
  RWebServicesEnv.properties, of each service)


Simple Integration Tests
------------------------

- These are simple integration tests and remote queries of services
  using a simple integration client -- the services respond to
  requests for information appropriately, and perform and return
  calculations as expected.
